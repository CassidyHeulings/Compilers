# Compilers
Labs and Projects for Compilers class
## How to build and run
Add your code to compile into {file for code to run}
### Build (MAC)
Must be run after every change.
Build from VSCode Terminal: Shift + Command + B
Build from VSCode Command Palette: Terminal -> Run Build Task
### Run
Run from VSCode terminal: ./Compiler/Compiler
Program output will be located in terminal.
Run from VSCode Run and Debug
Program output will be located in Debug Console.
